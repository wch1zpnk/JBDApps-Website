Description

Searchable clipboard history for everyday Mac work.

Everything Clipboard keeps a searchable history of what you copy on your Mac, so useful text, links, colors, files, and pasteboard content are easy to find again.

Instead of losing each copied item when the next one replaces it, Everything Clipboard stores a local history you can browse by pasteboard, content type, source app, or search. It is built for writing, research, support, development, design notes, and any workflow where you copy more than one thing at a time.

Key features

Built for fast finding, careful capture, and repeated paste workflows.

  • Use the first-run setup assistant to choose capture, privacy, and Direct Paste defaults.
  • Search clipboard history from a full app window or the fast Quick Paste window.
  • Use Command-number shortcuts in Quick Paste for the first ten visible results.
  • Monitor the standard General pasteboard, plus macOS Find, Font, and Ruler pasteboards.
  • Browse by content category, including text, links, code, colors, images, video, audio, files, rich content, pinned items, and secure items.
  • Pin important copied items so they stay easy to find.
  • Copy an older item back to the clipboard or paste it into the app you were using.
  • Remove exact duplicate history items while keeping one useful copy, with pinned items preferred when possible.
  • Build a Paste Stack when you need to paste several saved items in order.
  • Reorder a Paste Stack in a resizable configuration panel.
  • Clean copied text with tools for whitespace cleanup, tracking-link cleanup, Markdown links, and JSON formatting.
  • Recognize searchable text in copied images using on-device text recognition.
  • Choose how long history is kept and how many items are stored.
  • Get a warning when history is near its item limit before older unpinned items are cleaned up automatically.
  • Pause capture at any time.
  • Export and import passphrase-protected local history archives for backup or migration.

Privacy and Control

Everything Clipboard is local-first. Clipboard history is stored on your Mac, and the app gives you controls for what is captured, what is ignored, and when history is removed.

You can pause monitoring, choose which pasteboards are watched, ignore apps by bundle identifier, ignore matching text patterns, and require macOS authentication before revealing secure-looking text. The app includes default ignored apps and patterns for common password-manager and secret-looking content, and you can adjust those lists for your own workflow.

Quick Paste

Open Quick Paste when you want a keyboard-first way to find something you copied earlier. Search recent items, use arrow keys or Command-number shortcuts to move through results, and copy or paste the selected item without digging through a long history.

Paste Stack

Paste Stack helps when one copied item is not enough. Select saved items, add them to the stack, then paste each one in sequence. It is useful for repeated forms, support replies, research notes, and multi-step writing tasks.

Encrypted History Archives

Export a passphrase-protected archive when you want a local backup or migration copy of clipboard history. Secure item reveal text remains protected in Keychain and is not included in exported archives.

Designed for Mac

Native windows, toolbars, menus, settings, and Help keep detailed clipboard tools close at hand.
