Description

A clearer way to work with custom Voice Control commands.

Voice Command Atlas turns the custom commands already stored by macOS into a searchable list with plain details about each phrase, app scope, action type, status, and linked file.

It is built for people with large Voice Control command libraries who need to find commands quickly, spot problems, keep backups, and make deliberate changes without manually digging through a property list.

Key features

Search, understand, and maintain your command library.

  • Search custom commands and inspect the selected command's phrase, status, scope, action, target file, path, and modified date.
  • Open or reveal AppleScript files used by file-backed commands.
  • Add file, menu item, keyboard shortcut, and paste-text commands.
  • Edit a command's spoken phrase and app scope.
  • Enable, disable, or delete individual commands after an explicit action.
  • Test a saved command action and view a compact result.
  • Audit disabled commands, duplicates, critical issues, backup counts, setup, and compatibility.
  • Review command health for duplicate or similar phrases, stale references, missing files, and risky shortcuts.
  • Create automatic backups before command changes and restore a selected snapshot.
  • Use suggested profiles to manage groups by action type or app scope.
  • Export the current command file and compare another snapshot before restoring it.
  • Use a compact floating command-list window when you only need a quick reference.

Local Data and Backups

Voice Command Atlas works with the custom Voice Control command file already stored on your Mac. Command data, backups, exported snapshots, and test results stay local. The app creates a backup before actions that change command data.

Command Health

Use Health and Audit views to find repeated or similar spoken phrases, disabled commands, stale or unresolved file references, missing files, unknown actions, broad menu scopes, and risky keyboard shortcuts.

Focused Command Editing

Add common command types, adjust phrases and app scope, change status, refresh stale file references, or delete a command with clear controls and confirmation where needed.

Snapshot Review

Export the current command file or compare another snapshot with the live library before deciding whether to restore a full backup or apply selected changes.

Voice Command Atlas FAQ

Where does Voice Command Atlas get my commands?

It reads the custom Voice Control command file stored by macOS in your user Library. The app changes that file only after an explicit add, edit, delete, enable, disable, refresh, restore, or grouped-profile action.

Does my command data leave my Mac?

No. Command data, backups, snapshots, and command test results stay local to your Mac.

Is Voice Command Atlas available on the Mac App Store?

Not yet. Version 1.0 is being prepared for release, and no Mac App Store download link is currently available.
